# Markdown 格式速查表（完整版）

本文汇总 Markdown 所有常用格式，包含语法、示例和渲染效果，适用于文档编写、笔记整理、技术文档等场景。



***

## 一、文本基础样式



| 格式类型 | 语法                         | 示例                   | 渲染效果       |
| ---- | -------------------------- | -------------------- | ---------- |
| 加粗   | `**内容**` 或 `__内容__`        | `**加粗文本**`           | **加粗文本**   |
| 斜体   | `*内容*` 或 `_内容_`            | `*斜体文本*`             | *斜体文本*     |
| 加粗斜体 | `***内容***`                 | `***加粗斜体文本***`       | ***加粗斜体文本***   |
| 删除线  | `~~内容~~`                   | `~~删除线文本~~`          | ~~删除线文本~~  |
| 下划线  | `<u>内容</u>`                | `<u>下划线文本</u>`       | 下划线文本      |
| 高亮   | `==内容==`（GitHub/Typora 支持） | `==高亮文本==`           | == 高亮文本 == |
| 换行   | 行尾加 2 个空格 + 回车             | `第一行  （空格）<br>第二行`   | 第一行第二行     |
| 分段   | 空一行分隔                      | `第一段内容<br><br>第二段内容` | 第一段内容      |

第二段内容 |

## 二、标题层级



| 层级   | 语法          | 示例            |
| ---- | ----------- | ------------- |
| 一级标题 | `# 标题`      | `# 一级标题`      |
| 二级标题 | `## 标题`     | `## 二级标题`     |
| 三级标题 | `### 标题`    | `### 三级标题`    |
| 四级标题 | `#### 标题`   | `#### 四级标题`   |
| 五级标题 | `##### 标题`  | `##### 五级标题`  |
| 六级标题 | `###### 标题` | `###### 六级标题` |

> 效果：按 
>
> `#`
>
>  数量递减字体大小，一级标题最大，六级最小。

## 三、列表格式

### 1. 无序列表



| 语法符号 | 示例                                       | 渲染效果                   |
| ---- | ---------------------------------------- | ---------------------- |
| `-`  | `- 列表项1<br>- 列表项2<br>&nbsp;&nbsp;- 子列表项` | - 列表项 1- 列表项 2  - 子列表项 |
| `*`  | `* 列表项1<br>* 列表项2`                       | \* 列表项 1\* 列表项 2       |
| `+`  | `+ 列表项1<br>+ 列表项2`                       | + 列表项 1+ 列表项 2         |

### 2. 有序列表



| 语法    | 示例                                          | 渲染效果                      |
| ----- | ------------------------------------------- | ------------------------- |
| `数字.` | `1. 列表项1<br>2. 列表项2<br>&nbsp;&nbsp;1. 子列表项` | 1. 列表项 12. 列表项 2  1. 子列表项 |

### 3. 任务列表



| 状态  | 语法         | 示例           | 渲染效果        |
| --- | ---------- | ------------ | ----------- |
| 已完成 | `- [x] 内容` | `- [x] 完成任务` | - \[x] 完成任务 |
| 待完成 | `- [ ] 内容` | `- [ ] 待办任务` | - \[ ] 待办任务 |

## 四、链接与引用

### 1. 普通链接



| 语法                  | 示例                                      | 渲染效果                         |
| ------------------- | --------------------------------------- | ---------------------------- |
| `[链接文本](地址 "可选描述")` | `[GitHub](https://github.com "代码托管平台")` | [GitHub](https://github.com) |

### 2. 锚点链接（文档内跳转）



| 语法              | 示例                 | 说明                    |
| --------------- | ------------------ | --------------------- |
| `[跳转文本](#目标标题)` | `[跳转到一级标题](#一级标题)` | 目标标题需与原文一致，忽略 `#` 和空格 |

### 3. 引用



| 层级   | 语法       | 示例                               | 渲染效果                      |
| ---- | -------- | -------------------------------- | ------------------------- |
| 一级引用 | `> 内容`   | `> 一级引用`                         | > 一级引用                    |
| 二级引用 | `>> 内容`  | `> 一级引用<br>>>`二级引用 \`            | > 一级引用>>`二级引用`            |
| 三级引用 | `>>> 内容` | `> 一级引用<br>>>`二级引用`<br>>>> 三级引用` | > 一级引用>>`二级引用`>>> 三级引用 \` |

## 六、代码格式

### 1. 行内代码



| 语法           | 示例                                | 渲染效果                        |
| ------------ | --------------------------------- | --------------------------- |
| `` `代码内容` `` | `` 行内代码：`git commit -m "feat"` `` | 行内代码：`git commit -m "feat"` |

### 2. 代码块（带语法高亮）



| 语法               | 示例                                                                    | 渲染效果                                                                  |
| ---------------- | --------------------------------------------------------------------- | --------------------------------------------------------------------- |
| `语言<br>代码内容<br>` | `go<br>func main() {<br>&nbsp;&nbsp;fmt.Println("Hello MD")<br>}<br>` | `go<br>func main() {<br>&nbsp;&nbsp;fmt.Println("Hello MD")<br>}<br>` |

> 支持语言：go、java、python、javascript、sql、markdown 等，不指定语言则无高亮。

## 八、其他实用格式

### 1. 分割线



| 语法    | 示例                          | 渲染效果           |
| ----- | --------------------------- | -------------- |
| `---` | `内容1<br><br>---<br><br>内容2` | 内容 1---内容 2    |
| `***` | `内容1<br><br>***<br><br>内容2` | 内容 1\*\*\*内容 2 |
| `___` | `内容1<br><br>___<br><br>内容2` | 内容 1\_\_\_内容 2 |

### 2. 脚注



| 语法                           | 示例                                | 渲染效果                       |
| ---------------------------- | --------------------------------- | -------------------------- |
| `文本[^ID]<br><br>[^ID]: 脚注内容` | `需要说明的文本[^1]<br><br>[^1]: 脚注详细说明` | 需要说明的文本 \[^1]\[^1]: 脚注详细说明 |

### 3. 表情符号（Emoji）



| 语法          | 示例                       | 渲染效果        |
| ----------- | ------------------------ | ----------- |
| `:emoji代码:` | `:tada: 成功 :warning: 注意` | 🎉 成功 ⚠️ 注意 |

> 常用 Emoji 代码：🎉(
>
> `:tada:`
>
> )、✅(
>
> `:white_check_mark:`
>
> )、⚠️(
>
> `:warning:`
>
> )、ℹ️(
>
> `:information_source:`
>
> )

## 编辑工具推荐



* 本地编辑：Typora、Obsidian、VS Code（装 Markdown 插件）

* 在线编辑：GitHub、GitLab、Notion、语雀、飞书文档

* 核心特点：支持实时预览，语法自动补全，导出 PDF/HTML 等格式
